Usage

  • CAD, also known as Computer Aided Design, is a way of designing objects using computers.
  • We can use CAD software such as LibreCAD, Autodesk and OpenSCAD to design these objects.
  • CAD can be either 2D or 3D.
  • We can use different perspectives in 2D CAD such as an isometric view.

Functions

  • We can use functions in Lua to define a block of code
  • This block of code has to be called for it to be ran
  • To define a function, use the function keyword followed by the function's name then two parenthesis.
  • You should then create an indent and write your code in the function. Once completed, end the function with the end keyword.
  • Functions can be called (outside of the function) by writing the function's name followed by two parenthesis.
  • To define parameters in the functions, which you can use in the function's code, create undefined variables in the parenthesus
  • When the function is called then, put your variables values in the parenthesis.
  • Use the return keyword to return a value, but not print it.

    • Examples

    function sum(a, b)
     return a + b
    end

    function subtract_defined()
     print(5 - 4)
    end

    print(sum(2, 4))
    subtract_defined()

    Outputs

    6
    1

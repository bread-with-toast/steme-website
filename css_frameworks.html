CSS Frameworks

  • Frameworks in CSS can be used to speed up web design.
  • Examples of CSS Frameworks are: TailwindCSS, WingCSS and Bootstrap
  • TailwindCSS is different to WingCSS and Bootsrap as it us utility based.

    • Examples

    TailwindCSS - Set a h1's foreground colour to blue 300

    <h1 class="text-blue-300" >Hello!</h1>

    Bootstrap - Set a h1's foreground colour to red

    <h1 class="text-danger" >Hello!</h1>
  • Learn more at Bootsrap's website and TailwindCSS's website

CSS Colours

  • We can style our elements' colours using the color: attribute.
  • This styles the foreground/text colour of our element.
  • We can also use background-color: which styles the background colour of our element.
  • The value after the attributes can be a hex, RGB(A) or a pre-set colour.

    • Examples

    background-color: blue;

    color: #0000FF;

    color: rgb(0, 0, 255);

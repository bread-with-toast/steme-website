Directories

  • Directories, also known as folders, are digital "areas" where files can be stored.
  • To make a directory in the Linux command line, use the mkdir folder_name command (make directory).
  • To go to a directory, use the cd command followed by the directory's name (change directory).
  • To go to the home directory, use the cd ~ command.
  • To remove an empty directory, use the rm -d command. If there are files, use the rm -f command. Make sure to follow the command with the name of your directory.

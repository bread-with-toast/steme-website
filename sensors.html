Sensors

  • Sensors are electronics components which collect input.
  • We can then use that input to create an output, such as turn on an LED.
  • We can also connect sensors with a microcontroller and program the microcontroller so we can use the sensor.
  • Examples of sensors include:
    • Name Purpose
    HC-SR04 (Ultrasonic Sensor) To find out how many ms it took for a sound to hit an object.
    IR sensor To detect IR.
    Fire detection sensor To detect fire.

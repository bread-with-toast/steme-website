If, Else, Else If

  • We can use an if statement to add more logic into our programs
  • The syntax of an if command goes like:
    if condition then
     code
    end
  • To add an else statement, we can use the else keyword. For an elseif, use the same syntax as an if command.

    • Note

    Here is a list of the operators you can use in an if or elseif statement.

    Symbol Meaning
    == Equal to
    != Not equal to
    x < y y greater than x
    x > y x greater than y
    x >= y x greater than or equal to y
    x <= y y greater than or equal to x

    Examples

    x = 0
    if x == 1 then
     print("hi")
    else
     print("hello")
    end

    y = "hello"
    if y != "hi" then
     print("y is not hi.")
    elseif y == "pen"
     print("y is equal to pen.")
    end

    Expected Outputs

    hello
    y is not hi.

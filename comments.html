Comments

  • Commenting your code can be a useful way to explain what lines of your code do.
  • You should not have to comment every line of your code as your code should be clear.

    • Bad Example (Lua)

    print("hello world") ~~ Prints "hello world" x = 3 ~~ Sets a variable called x to 3 print(x) ~~ Prints x

    Good Example (Lua)

    print("hello world") x = 3 ~~ X defined to print it print(x)

HTML5 Images & Links

  • HTML can also be used to show images.
  • To show images, use the <img>
  • An image will need a source attribute. This tells it what to display.
  • In the end, an image will look like: <img src="">

  • If you want to show a link in HTML, use the <a></a> tags.
  • You'll need the href attribute to use links.
  • In the end, a link will look like: <a href="link">text</a>

    • Accessibility Alert!

    To make the internet more accessible, you can also use the alt="" attribute for images.

    Consider adding a caption to your image aswell.

Buzzers

  • Buzzers are electronic components which can be used to make sound.
  • There are different types of buzzers, but the two mentioned in this module are called piezoelectric and magnetic buzzers.
  • Piezoelectric buzzers use piezoelectric ceramics, which is a material that changes forces into an electrical signal, while magnetic buzzers use a metal plate.
  • Buzzers are also polarised.

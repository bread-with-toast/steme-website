Tables

  • In Lua, tables can be used to define a set of data.
  • Tables are defined like a variable, except, the value changes to curly braces.
  • Then, we can define the different variables in a table.
  • Tables in Lua use dot notation, which means to refer to a value, we use a dot.

    • Example

    book1 = {}
    book1.pages = 32
    book1.words = 600
    print(book1.pages)

    Output

    32

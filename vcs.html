Version Control Systems

  • Version Control Systems are a way to control the versions of your programs.
  • Examples of Version Control Systems (VCSs) are: Git, Mercurial and SVN.
  • A useful feature of these VCSs is the ability to create branches. Branches allow you to have different sets of code and then the ability to merge them together, so you don't destroy your main program.
  • We can also connect these local repositories to remote repositories, which are hosted on the internet.

    • Examples of Git commands

  • git init
    • Initialises a local Git repository
  • git commit -m ""
    • Creates a new commit (change) with the message.
  • git log
    • Views the commits made to the program.

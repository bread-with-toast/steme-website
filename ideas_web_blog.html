Website Ideas

If you've been learning web design, I've created a list of projects you can try to make! Have a go at them!

1) Social Media Profile Page

Design a simple profile page which shows how many followers, viems and likes a person has. Use a grey profile picture, or any colour you want. Try adding in a few videos you for a challenge!

2) E-Commerce Page

Design an e-commerce page which shows a list of items you can buy. It should show their cost, a title, a short description and a picture of the item. For a challenge, try designing the page for a product and include a full description and a button to purchase the item.

3) A Landing Page

Design a landing page for whatever you want. Try making it stand out.

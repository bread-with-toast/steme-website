OpenSCAD Basics & Colours

  • To generate a cube in OpenSCAD, simply use the cube(x); command.
  • x is the value of the sides.
  • This also works with a sphere and cylinder, where x is the size of the sphere and height of the cylinder.
  • To create shapes with different lengths, you can use square brackets followed by the lengths.
  • For colours, you can use the color([]) command with RGB values seperated by commas from 0-1.

    • Examples

    cube([20,50,30]);
  • Generates a cube with the dimensions 20*50*30

    • color([0, 1, 0]) sphere(8);
  • Generates a green sphere with the size of 8.

HTML5 Lists & Buttons

  • We can also display lists in HTML.
  • You are looking at a list right now.
  • This list is called an unordered list and is defined with the <ul></ul> tags.
  • Unordered lists use bullet points.
  • The other type of list is called an ordered list and uses numbers. It uses the <ol></ol> tags.
  • To display items in a list, use the list item tags (<li></li>)

    • Example (ordered list):

    1. Buy food
    2. Cook food
    3. Eat food

  • To make buttons, just use the <button></button tags.>

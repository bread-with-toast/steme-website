CSS Colours

  • In CSS, we can use the # symbol to style elements with a specific ID.
  • We can use the . symbol to style specific classes
  • After the selector, type in curly braces and put your styles in there.

    • Examples

    #logo { color: green; }

    .large_font { font-size: 500px; }

    h1 { opacity: 0.5; }

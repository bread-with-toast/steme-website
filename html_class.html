HTML5 Divs, Classes & IDs

  • HTML Divs can be used to seperate your HTML source code for your websites websites. Use the <div> </div> tags to create a div.
  • HTML Classes can be used to style more than one element by using the class="" attribute in an element.
  • HTML IDs can be added to an element by adding the id="" attribute. IDs also help style elements.

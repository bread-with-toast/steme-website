File Permissions

  • We can use the chmod command to change file permissions in the Linux terminal.
  • The syntax of the command is:
  • chmod [people]-[permission] [file]
  • For example, chmod -r hello.txt adds read permissions to the owner of the file hello.txt.
  • The [people] parameter is optional.

    • Table For [permission]

    Permission Meaning
    -r Read
    -w Write
    -x Run

    Table For [people]

    People Meaning
    o Others
    g Group
    a All
